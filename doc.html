Implementace

Implementován byl Use-case diagram z předmětů IUS a IDS, nebyly provedeny žádné změny. Stejně databáze se shoduje z ER diagram z předmětů IUS a IDS

Při implementaci nebyly použity žádné frameworky, vše bylo vytvořené pomocí jazyků HTML, CSS a MySQL

Výpis PHP skriptů a jejich popis:

  • admin.php - mazání a přidání zaměstnance
  • cart.php - nákupní košík, odstranění kostýmů z košíku, vynulování celého košíku
  • client.php - seznam výpůjček uživatele
  • common.php - funkce pro vytvoření hlavičky a zápatí, funkce pro filtrování
  • costumes.php - zobrazení dostupných kostýmů
  • db_init.php - inicializace databaze
  • employee.php - seznam všech kostýmů a doplňků, mazání/přidávání kostýmů a doplňků
  • index.php - úvodní stránka
  • login.php - přihlášení a odhlášení do systému
  • product.php - zobrazení kostýmů a doplňku
  • registration.php - registrace nového uživatele
  • reservation.php - rezervacia kostýmů

Instalace

  • Pro správné spuštění systému třeba všechny adresáře a soubory uložit na server do cílové složky
  • Importujte celou databázi ze souboru db.sql například přes system phpMyAdmin
  • Aplikace vyžaduje PHP 5 a MySQL 5
